# model-verse notebooks

Starter notebooks for exploring [PyPSA](https://pypsa.org) community models listed on
the [modelverse](https://modelverse.bayesian.energy). Each notebook downloads the model's
published PyPSA network, draws the grid, and uses PyPSA's statistics module to chart
installed capacity and dispatch — open it in Google Colab and run all cells.
