# model-verse notebooks

Starter notebooks for exploring [PyPSA](https://pypsa.org) community models listed on
the [modelverse](https://modelverse.bayesian.energy). Each notebook downloads one of the
model's published PyPSA networks, draws the grid, and uses PyPSA's statistics module to
chart installed capacity and dispatch — open it in Google Colab and run all cells.

There is **one notebook per published network** (`.nc` file), so a model with several
variants or vintages has a notebook for each.
